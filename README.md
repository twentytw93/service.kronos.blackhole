# Kronos Blackhole

Kronos Blackhole is a program add-on for **Kodi 21.x (Omega)** designed for **LibreELEC** and **Windows** systems. 
It blocks unwanted domains at the DNS level to prevent trackers, telemetry, and intrusive network requests.

---

## Features
- **DNS-Level Blocking** – Intercepts and blocks tracking and telemetry domains. 
- **Tracker Protection** – Stops known tracking endpoints before connection. 
- **Silent Background Operation** – Runs quietly with minimal system impact. 
- **Lightweight Filtering Engine** – Fast lookup and low-latency blocking. 
- **Fail-Safe Design** – Designed to avoid system instability or network lockups. 
- **Privacy-Focused** – Enhances user privacy without affecting normal connectivity.

---

## Compatibility
- Kodi **21 Omega** 
- LibreELEC 12.0.2+ 
- Windows 10/11 
- Arctic Zephyr Reloaded skin 
- Python 3 (bundled with Kodi)
